CSS Cheat-sheet

Here are some notes and things I've learned along the way to help me with styling my HTML.
Some aren't commonly used, so this is a perfect sheet to reference if I don't remember the styling commands.


Position: absolute, block, inline, inline-block
Colour: rgb, rgba(0,0,0,0)
Text-transform: capitalize, uppercase, lowercase
Float: left, right, center
Width/Height: vh, vw, px, calc(var(--divHeight)), calc(var(--divWidth)) - (must have :root{ --GameWidth: #vh; --GameHeight: #vh; } in the CSS file to make use of divHeight/Width)
Center an Object: left: 50%; transform: translate(-50%);
Grid-template-columns: auto (an 'auto' for each column I want to create)
Grid-gap: px
Border: colour, px, solid, dotted, dashed, double, none, hidden
Border-radius: percent (50% to make a circle)
Line-height: number (default = 1)
